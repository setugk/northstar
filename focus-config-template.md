## Reference

**Time options:** 15 min, 30 min, 60 min (1 hour), 120 min (2 hours)

**Accepted time formats:** You can write times in any of these forms:
- `30 min`, `30m`, `30min`, `30 minutes`
- `1 hour`, `1h`, `1hr`, `2 hours`, `2h`
- The value must resolve to one of: 15, 30, 60, or 120 minutes

**To add a task:** Copy this block under the relevant goal:
```
#### X.X [Task title]
- **Time:** 30 min
- **Link:** [Label](https://url.com)
- **Notes:**
```

**To add a goal:** Copy this block below the last goal:
```
### X.0 [Goal Title]

#### X.1 [First task]
- **Time:** 30 min
- **Notes:**
```

**Rules:**
- Task and goal IDs are auto-renumbered on upload, so don't worry about keeping them in sync
- Link and Notes are optional
